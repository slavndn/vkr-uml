# UML‑набор диаграмм для ВКР + прототип интерфейса

Тема ВКР: **Разработка веб‑приложения для учета спортивных достижений обучающихся на уроках физической культуры в школе**

## Что сделано
- Набор UML‑диаграмм в PlantUML, раскрывающих **статическую** и **динамическую** сущности продукта:
  - Use Case
  - Class Diagram
  - Sequence Diagram
  - State Diagram
  - Activity Diagram
- Программный код (TypeScript), подготовленный на основе диаграммы классов
- Краткий отчет по выполнению задания
- Wireframe (прототип интерфейса) в Mermaid (Markdown)

## Структура репозитория
- `diagrams/`
  - `use-case.puml` — диаграмма вариантов использования
  - `classes.puml` — диаграмма классов
  - `sequence-add-achievement.puml` — диаграмма последовательности (сценарий «внесение достижения»)
  - `state-achievement-record.puml` — диаграмма состояний (жизненный цикл записи достижения)
  - `activity-record-achievement.puml` — диаграмма деятельности (процесс фиксации результата)
- `src/domain/` — код доменных моделей (TypeScript), соответствующий диаграмме классов
- `docs/`
  - `report.md` — отчет/описание предметной области и подхода
  - `wireframe.md` — прототип интерфейса (Mermaid)

## Как открыть диаграммы
### Вариант A: VS Code
1. Установите расширение **PlantUML**.
2. Откройте любой файл из `diagrams/*.puml`.
3. Запустите предпросмотр диаграммы (Preview).

### Вариант B: JetBrains IDE
1. Установите плагин **PlantUML integration**.
2. Откройте `diagrams/*.puml` и используйте Preview.

## Как открыть wireframe (Mermaid)
Файл `docs/wireframe.md` содержит блок ` ```mermaid ` — его можно смотреть в GitHub (рендерится автоматически) или в редакторе с поддержкой Mermaid.
